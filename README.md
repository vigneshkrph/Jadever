# Static Angular Storefront

This project is a static Angular landing page inspired by a premium retail marketplace layout. It uses a single-page structure with a hero area, category cards, featured collections, trust signals, and a contact block.

## Development

Install dependencies and start the app with:

```bash
npm install
npm start
```

## Build

Create a production build with:

```bash
npm run build
```

## Notes

The current design is intentionally static. Replace the placeholder copy, contact details, and visual themes with your own brand content when you are ready.
